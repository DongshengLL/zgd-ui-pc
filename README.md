## Usage

1.  Install package [zgd-ui-pc]
    ```sh
    npm i -D zgd-ui-pc
    ```
2.  Get Started

        ```vue
        <template>
            <div id="app">
                <s-button type="primary">点击报名</s-button>
            </div>
        </template>

        <script>
        import { Button } from 'zgd-ui-pc';
        import 'zgd-ui-pc/dist/sakura.css';
        export default {
            name: "app",
            components: {
                "s-button": Button,
            },
        };
        </script>
        ```
